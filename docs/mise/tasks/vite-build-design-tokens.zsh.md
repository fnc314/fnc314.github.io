## `vite:build:design-tokens`

Invokes `vite` within `design-tokens`


- Depends: dev-ex:tools:style-dictionary

- **Usage**: `vite:build:design-tokens [mode]`
- **Aliases**: `v:b:d-t`

### Arguments

#### `[mode]`

The `mode` flag for `vite`

**Choices:**

- `development`
- `production`

**Default:** `development`
