## `dev-ex:tools:style-dictionary`

Invokes `style-dictionary` with `./.config/style-dictionary/config.ts`


- **Usage**: `dev-ex:tools:style-dictionary [mode, m]`
- **Aliases**: `dx:t:style-dictionary`

### Arguments

#### `[mode, m]`

The `mode` flag for `vite`

**Choices:**

- `development`
- `production`

**Default:** `development`
