## `dev-ex:refresh`

- **Usage**: `dev-ex:refresh [logstep]`
- **Aliases**: `dxr`

Run dx-refresh

### Arguments

#### `[logstep]`

Creates a .log file timestamped by execution point in ./logs/&lt;choice>/&lt;timestamp>.log

**Choices:**

- `none`
- `log-all`
- `mise-tasks`
- `mise-tasks-design-tokens`
- `typescript-types`
- `pwrs/cem`
- `cem-analyze`
- `wca`
- `typedoc`

**Default:** `none`
