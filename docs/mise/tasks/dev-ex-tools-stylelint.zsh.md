## `dev-ex:tools:stylelint`

Runs `stylelint`


- **Usage**: `dev-ex:tools:stylelint [FLAGS]`
- **Aliases**: `dx:t:stylelint`

### Flags

#### `--fix`

Passes `--fix` to `stylelint`

**Default:** `false`

#### `--analyze`

Passes `--cei` to `stylelint`

**Default:** `false`

#### `--log`

Creates a `HHMMSS.log` file at `./logs/stylelint/{YYYYMMDD}

**Default:** `false`

#### `--verbose`

Passes `-f verbose` to `stylelint`

**Default:** `false`

#### `--config-inspector`

Runs `stylelint-config-inspector` instead

**Default:** `false`
